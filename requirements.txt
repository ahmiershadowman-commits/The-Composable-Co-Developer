# The Composable Co-Developer Marketplace Runtime

Python 3.10+ required.

## Installation

```bash
# Create virtual environment (optional but recommended)
python -m venv venv
venv\Scripts\activate  # Windows

# Install dependencies
pip install -r requirements.txt
```

## Usage

### Run Test Suite

```bash
python -m pytest tests -v
```

### Run Vertical Slice

```bash
python tools/run_vertical_slice.py
```

### Validate Bundle

```bash
python tools/validate_bundle.py
```

### Adding New Pipelines

1. Create pipeline directory under appropriate family
2. Add `pipeline.yaml` spec
3. Implement executor method
4. Add tests
5. Update family route map

### Running Tests

```bash
# All tests
python -m pytest tests -v

# Specific test module
python -m pytest tests/runtime/test_registry_load.py -v

# With coverage
python -m pytest tests --cov=. --cov-report=html
```
