# Shared Component Packets

## Trace
Traveling metacognitive controller.
Applies smallest-sufficient intervention before escalation.

## Lever
Evaluator / escalation / commitment authority.
Invoked when local supervision is insufficient.

## Residue
Investigative lens library.
Used when the visible account feels smoothed, incomplete, suspicious, or structurally odd.

## Primitives
Low-cost reusable interventions.

## Operators
Heavier local transformations that are still cheaper than family reroute.

## Feedback loops
Family and global loop topology.
Must stay consistent with selectors, route maps, and target grammar.
